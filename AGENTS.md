# Tinygrad Lessons Learned
- **Autogen Clang Verification**: When executing `pytest` across the `tinygrad` codebase, `LIBCLANG_PATH=/usr/lib/llvm-19/lib/libclang.so` must be explicitly exported. Otherwise, `autogen` tests will fail with an `AttributeError` failing to load `libclang.so`.
- **CStyleLanguage Context Initialization**: When extracting dynamically generated attributes (like `dtcm_bump`) from the `CoralNPURenderer` context inside `CStyleLanguage` rewrites, always use `getattr(ctx, 'dtcm_bump', 4096)`. Manual `render_kernel` invocations in tests bypass the standard `render()` initialization, causing silent `AttributeError` failures if accessed directly.
- **Register Spilling Liveness**: When implementing deferred register spilling, `u.src` iterates over the original UOp graph. You must track and apply `replacements` mapping the original UOp to its newly `LOAD`ed active register representation to ensure topological references correctly map to the spill buffer instead of the original dead register.
